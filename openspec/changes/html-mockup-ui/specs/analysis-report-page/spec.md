## ADDED Requirements

### Requirement: 分析报告页面布局
页面 SHALL 包含顶部 Header、面包屑导航、报告头部（标题+关键词+状态徽章）、三个分析区域（研究空白、Base 论文推荐、创新点）。

#### Scenario: 渲染分析报告
- **WHEN** 用户访问 /projects/{id}/analysis 且分析已完成
- **THEN** 显示 24px 标题、关键词和「分析完成」徽章，以及三个编号区域

### Requirement: 研究空白网格
页面 SHALL 以 2×2 网格展示研究空白，每个卡片含分类标签、描述、支撑证据、关联论文。

#### Scenario: 渲染研究空白卡片
- **WHEN** 分析结果包含多个研究空白
- **THEN** 每个卡片显示分类标签（方法学局限/矛盾结论/未探索领域/数据集不足），对应不同配色

### Requirement: Base 论文推荐列表
页面 SHALL 以纵向列表展示推荐论文，每项含排名、标题、作者、推荐理由、创新方向、操作按钮。

#### Scenario: 渲染推荐论文卡片
- **WHEN** 存在推荐的 Base 论文
- **THEN** 每篇论文显示编号徽章（1/2/3）、标题、作者、推荐理由灰色背景框、创新方向紫色文字、「设为 Base 论文」按钮

### Requirement: 创新点网格
页面 SHALL 以 3 列网格展示创新点，每张卡片含难度标签、描述、关联论文、对应研究空白。

#### Scenario: 渲染创新点卡片
- **WHEN** 存在创新点分析结果
- **THEN** 每张卡片顶部有难度标签（低/中/高）和类型标签，卡片顶部有对应难度的彩色横条（绿/橙/红）
